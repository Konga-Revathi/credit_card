# Credit Card Fraud Detection

## Overview

This project uses Machine Learning to detect whether a credit card transaction is fraudulent or legitimate. The model is trained using historical transaction data and helps identify fraud transactions accurately.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- SMOTE

## Dataset

Dataset: Credit Card Fraud Detection Dataset

Source:
https://www.kaggle.com/mlg-ulb/creditcardfraud

## Algorithms Used

- Logistic Regression
- Decision Tree Classifier

## Project Steps

1. Load Dataset
2. Data Preprocessing
3. Handle Imbalanced Data using SMOTE
4. Train Machine Learning Models
5. Evaluate Performance
6. Predict Fraudulent Transactions

## Results

- Successfully detected fraudulent transactions.
- Compared Logistic Regression and Decision Tree models.
- Evaluated using Accuracy, Precision, Recall, and F1-Score.
